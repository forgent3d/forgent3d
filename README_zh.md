# Forgent3D
> 代码就是模型

[English](README.md)

Forgent3D 是一个将大语言模型（LLM）生成的代码转化为三维模型，并进行实时预览的桌面工具（基于 Electron）。

## ✨ 核心思路
打破传统的建模门槛，底层内核基于强大的 **build123d**，你可以自然地结合 **Cursor**、**Claude**、**Code** 或 **Codex** 等 AI 编程助手来生成和编写模型代码。Forgent3D 负责将这些代码即时渲染，让你直观地看到最终的 3D 效果。
